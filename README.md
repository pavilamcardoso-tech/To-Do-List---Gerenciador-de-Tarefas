📝 To-Do List — Gerenciador de Tarefas:
Aplicação web fullstack de gerenciamento de tarefas com autenticação de usuários.

🚀 Tecnologias
Frontend: React, Vite, Axios
Backend: Node.js, Express
Banco de dados: PostgreSQL

✨ Funcionalidades
Cadastro e login de usuários
Adicionar, concluir e remover tarefas
Filtrar tarefas por status (todas, ativas, concluídas)
Dados salvos no banco de dados por usuário
